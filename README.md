# KenshiLimbs

RE_Kenshi/KenshiLib plugins for fixing and extending Kenshi's robotic limb system.

## Mods

- [`The Limbless (Type 2)`](The%20Limbless%20(Type%202)/README.md) - bundles the "The Limbless"
  race mod with `RobotLimbRaceLock.dll`, a plugin that fixes vanilla's FCS-configured robotic limb
  race restrictions (which are silently ignored by the base game engine).
